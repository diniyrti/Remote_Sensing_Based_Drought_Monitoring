# Remote Sensing–Based Drought Monitoring | Jeneponto, Indonesia (2015–2024)

## Overview
[cite_start]This repository contains the implementation of a **Synthetic Drought Index (SDI)**[cite: 81]. [cite_start]The method integrates multiple remote sensing indicators to provide a comprehensive assessment of drought severity in Jeneponto Regency[cite: 89, 93].

## Methodology
[cite_start]The SDI is calculated using **Principal Component Analysis (PCA)** [cite: 23, 270] by merging:
* [cite_start]**VCI** (Vegetation Condition Index) - Derived from Landsat 8[cite: 78, 211].
* [cite_start]**TCI** (Temperature Condition Index) - Derived from Land Surface Temperature[cite: 77, 235].
* [cite_start]**PCI** (Precipitation Condition Index) - Derived from CHIRPS/TRMM data[cite: 76, 254].

## Data Sources
* [cite_start]**Satellite Imagery**: Landsat 8 OLI/TIRS[cite: 153].
* **Precipitation**: CHIRPS (Climate Hazards Group InfraRed Precipitation with Station data).
